# What?

We have investigations which ask concrete questions which are close-ended and aim to be resolvable. Explorations are open-ended and generate leads on the other hand.

We use indices for the explorations to be able to duplicate an exploration title. Because in time, we may revisit an area with new knowledge or at a very different time, so we increment the index.
