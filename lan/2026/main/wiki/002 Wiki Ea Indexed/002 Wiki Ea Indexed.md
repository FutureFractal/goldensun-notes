# What?

Many notes for reverse engineering projects can be taken as indexing an effective address for the game. Here we provide documentation layers, each indexing by an ea so that interested readers can filter for the kind of content needed for that ea.

In bn6f, I did a bunch of experiments such as finding out what happens if a feature was disabled. This could be included here as one interpretation of the game items.
